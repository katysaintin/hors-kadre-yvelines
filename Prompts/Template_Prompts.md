## TEMPLATE 2 — Article #25
## "Et si ça vous a touché — qu'est-ce que ça change ?"

```
# ARTICLE HORS KADRE #25

## CONTEXTE
Un texte fictif sur une prof à la retraite circule 
sur les réseaux. Les gens sont touchés.
Des commentaires reçoivent : "Rédigé par une IA !!"
Comme si ça annulait l'émotion ressentie.

Commentaire reçu sur mon article Poisson (Mediapart) :
"Sympa la formulation et le rythme d'écriture, 
même si optimisé par l'IA. Ca m'a donné envie 
de lire jusqu'au bout."

Ma réponse — qui EST la thèse :
"Lire jusqu'au bout c'est le but. 
Juste faire passer mes idées — donc ça le touche."

## QUESTION CENTRALE
Si ça vous a donné envie de lire jusqu'au bout —
qu'est-ce que ça change que l'IA ait aidé ?

## ANGLES
1. La prof fictive — émotion réelle malgré l'outil
2. L'art a toujours utilisé des outils
   (orchestre, plume d'oie, Pro Tools)
3. Mon commentaire poisson — preuve concrète
4. Syndrome de l'imposteur version HPI
   → "un HPI ne devrait pas avoir besoin d'IA"
   → case dans la case
   → culpabilité imposée par une communauté
      censée comprendre
5. Je partage les lauriers sans jalousie
6. Chute : "Vous avez lu jusqu'au bout.
   C'est tout ce que je voulais.
   Le reste — c'est votre débat."

## TON
Satirique, humour discret, accessible grand public

## CONTRAINTES MEDIAPART
Titre 80 car. / Chapô 400 car. / Mots-clés / Prompt image
Numéro : 25 — Déclinable Medium : oui
```

## TEMPLATE 3 — Article #25bis
## "Le système a un switch/case. Le cerveau n'a pas de break."

```
# ARTICLE HORS KADRE #25bis

## CONCEPT CENTRAL
Expliquer les limites du système de diagnostic 
neurodiversité (MDPH, spécialistes, missions handicap)
via une analogie informatique.

Le système actuel = switch/case sans break :
switch (profil) {
    case "TDAH":  → traitement A
    case "HPI":   → traitement B
    default:      → "vous n'existez pas"
}

Ce qu'il devrait faire = if(conditions multiples) :
if (memoire_travail < seuil
    && vitesse_traitement > seuil
    && compensation == true
    && genre == "F") {
    → accompagnement adapté
}

## ANGLES
1. Le bug du système — métaphore portier de boîte
2. Le switch/case en action (MDPH, DSM-5)
3. Tomber dans le default
   (HPI + TDAH + femme + réussite = "impossible")
4. Le switch/case dans le switch/case
   (spécialistes neurodiversité qui recatalogent)
5. Ce que le code devrait faire
6. Chute : "En informatique, un code qui plante 
   à chaque cas non prévu, on appelle ça un bug.
   En médecine, on appelle ça le système."

## CIBLE
Geeks d'abord → RH → grand public
Vulgarisation obligatoire pour non-informaticiens

## VÉCU
Parcours diagnostic personnel anonymisé
RQTH obtenue après 1 an / burnout décembre 2025

## CONTRAINTES MEDIAPART
Titre 80 car. / Chapô 400 car. / Mots-clés / Prompt image
Numéro : 25bis — Déclinable Medium : oui
```
