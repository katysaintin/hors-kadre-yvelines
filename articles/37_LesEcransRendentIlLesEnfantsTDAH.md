# Idée article Mediapart — « Les écrans rendent-ils nos enfants TDAH ? »

## Angle central

Ou pourquoi faut il être dans une case pour être accompagné.
Suite de l'article Mme HorsKase
Refuser les raccourcis actuels autour :

* des écrans,
* du TDAH,
* de l’éducation positive,
* des crises adolescentes,
* et des discours anxiogènes du web.

Positionnement personnel fort :

* adulte TDAH diagnostiquée tardivement + RQTH,
* mère concernée (TDAH + TSA),
* ingénieure / chercheure en neurosciences (Neurospin / ISEULT),
* formée CNV + Faber & Mazlish,
* défense des personnes neuroatypiques MAIS aussi des neurotypiques invisibles.

Idée clé :

> Les outils humains respectueux ne devraient pas être réservés aux personnes ayant une “case”.

---

# Thèse principale

Le problème des cases n’est pas qu’elles existent.

Le problème est qu’elles sont devenues la seule monnaie d’échange pour accéder :

* à la compréhension,
* à l’aide,
* aux adaptations,
* à l’écoute,
* à des environnements plus humains.

Aujourd’hui :

* un enfant “ordinaire” qui souffre des écrans → “manque de cadre”
* un enfant TDAH → compréhension partielle
* un ado mal → “c’est l’âge”
* un adulte épuisé → “tout le monde est fatigué”
* un aidant naturel → invisible

Il faut toujours :

* une case,
* un diagnostic,
* une catégorie validée,
  pour légitimer des besoins profondément humains.

---

# Positionnement subtil à conserver

Ne PAS nier :

* le handicap,
* les vrais TDAH,
* les besoins d’adaptation,
* les souffrances neurodéveloppementales.

Au contraire :
le vécu personnel donne la légitimité pour dire :

> “Je sais ce qu’est un vrai TDAH. Justement pour cette raison, je refuse les raccourcis.”

L’objectif n’est PAS :

* “tout le monde est TDAH”
  NI
* “le TDAH n’existe pas”

Mais :

> certains mécanismes concernent tous les cerveaux humains, avec des intensités différentes.

---

# Point fort philosophique

Les écrans exploitent :

* la dopamine,
* la recherche de nouveauté,
* la gratification immédiate,
* la fatigue exécutive,
* les vulnérabilités attentionnelles humaines universelles.

Le TDAH peut amplifier ces difficultés.
Mais les enfants neurotypiques ont eux aussi :

* un cortex préfrontal immature,
* une régulation émotionnelle incomplète,
* une faible inhibition,
* une forte sensibilité à la récompense immédiate.

Donc :

* les conseils proposés aux enfants TDAH sont souvent de bons conseils éducatifs pour TOUS.

---

# Critique des dérives web / clickbait

Le web actuel récompense :

* l’anxiété,
* les oppositions binaires,
* les titres simplistes,
* les contenus émotionnels.

Exemples :

* “Les écrans détruisent le cerveau”
* “L’éducation positive crée des enfants rois”
* “La Suède est devenue laxiste”
* “Tout le monde est TDAH”
* “Les ados n’ont plus de limites”

Refuser ce fonctionnement.
Refuser les slogans.

Positionnement Hors Kadre :

> complexifier plutôt que simplifier.

---

# Partie importante sur la CNV / Faber

Idée centrale :

> Bienveillance ≠ absence de limites.

Développer :

* fermeté bienveillante,
* autorité relationnelle,
* cadre sans humiliation,
* empathie sans permissivité.

Idée forte :

> L’éducation positive authentique est exigeante parce qu’elle oblige aussi l’adulte à travailler sur lui-même.

Lien avec vécu TDAH :

* régulation émotionnelle parentale,
* fatigue exécutive,
* surcharge mentale,
* maintien du cadre malgré ses propres difficultés.

---

# Pont avec “Mme HorsKase”

Réutiliser l’idée :

> “Le problème des cases — ce n’est pas qu’elles existent. C’est qu’elles sont devenues la seule monnaie d’échange pour accéder à ce dont tu as besoin.”

Possible continuité philosophique :

* PAP,
* RQTH,
* adaptations scolaires,
* compréhension émotionnelle,
* droit à des environnements moins violents.

Question :
Pourquoi faut-il souffrir suffisamment ET dans la bonne catégorie pour avoir droit à des outils humains de base ?

---

# Pont avec l’article sur les aidants naturels

Montrer que :

* les aidants neurotypiques souffrent aussi,
* les parents “sans case” existent,
* les conjoints existent,
* les fratries existent.

Refus d’un discours :

* “si tu n’as pas de diagnostic, ta souffrance compte moins.”

---

# Positionnement final

Ne pas opposer :

* neurotypiques / neuroatypiques,
* éducation / neurobiologie,
* fermeté / bienveillance.

Mais montrer :

* continuités humaines,
* gradients,
* vulnérabilités universelles,
* et nécessité d’un cadre plus humain pour tous.
