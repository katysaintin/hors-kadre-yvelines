# SAV Mme Bénévole bonjour

## Spin-off du “Chat dans le transfo”

---

# Idée générale

Après :

* le support industriel ;
* les astreintes SCADA ;
* les écrans rouges ;
* et le chat dans le transformateur ;

découverte d’un autre système complexe :
les parents d’élèves.

Même logique :

* utilisateurs perdus ;
* documentation jamais lue ;
* workflows ignorés ;
* demandes hors périmètre ;
* support émotionnel ;
* et humains “tampons” qui absorbent tout.

---

# Concept central

Les personnes :

* compétentes ;
* aidantes ;
* pédagogues ;
* disponibles ;
* et capables de résoudre des problèmes hors périmètre ;

deviennent progressivement :

## les interfaces humaines du système.

Les utilisateurs :

* ne cherchent plus ;
* ne lisent plus ;
* ne contactent plus le bon interlocuteur ;
* ils appellent directement :
  la personne qui sait.

---

# Évènement déclencheur du spin-off

Mail reçu d’une maman :

> “Bonjour Katy, tu connais la date butoir pour remplir les spécialités et le choix du lycée dans EduConnect ? Merci beaucoup.”

Réflexe immédiat :
aller chercher l’information.

Puis découverte :

* la réponse était déjà dans l’ENT ;
* envoyée officiellement par le lycée ;
* avec date limite clairement indiquée.

Conclusion intérieure :

> “SAV Mme Bénévole bonjour.”

---

# Déclic narratif

Prise de conscience :
les représentants de parents deviennent eux aussi :

## du support niveau 1 humain.

Même logique que :

* le support informatique ;
* les hotlines ;
* les accueils ;
* les secrétariats ;
* les urgences ;
* les professeurs principaux.

Le front visible absorbe :

* les questions ;
* les frustrations ;
* les oublis ;
* les angoisses ;
* et parfois :
  la flemme cognitive des utilisateurs.

---

# Anecdote fondatrice

Discussion avec mon ancien chef au synchrotron SOLEIL.

Contexte :

* statistiques Jira ;
* pics d’appels utilisateurs pendant mes semaines d’astreinte.

Question :

> “Mais pourquoi quand c’est moi les utilisateurs se lâchent ?!”

Réponse du chef :

> “Parce que tu es la seule du groupe à dépanner les utilisateurs corps et âme. Même hors périmètre.”

Puis :

> “Du coup ils ne réfléchissent même plus. Ils t’appellent.”

* absorbent spontanément les problèmes des autres ;
* deviennent les interfaces naturelles des systèmes ;
* et finissent parfois :
  à porter des charges qui ne devraient pas reposer sur une seule personne.

# Angle émotionnel

Article :

* drôle ;
* auto-dérisoire ;
* pas plaintif ;
* pas victimaire.

Objectif :
faire rire les gens qui travaillent :

* dans le support ;
* l’accueil ;
* le soin ;
* l’enseignement ;
* le bénévolat ;
* ou toute fonction “tampon” humaine.

Avec derrière :
une vraie réflexion sur :

* la charge cognitive ;
* la délégation mentale ;
* et les humains qui simplifient le monde pour les autres.
