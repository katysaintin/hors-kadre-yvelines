## L'IDÉE DE L'ARTICLE
Expliquer les limites du système de diagnostic et de prise en 
charge de la neurodiversité (MDPH, missions handicap, spécialistes)
via une analogie informatique : le switch/case sans break.

### L'analogie centrale :

Le système actuel fonctionne comme un switch/case :
switch (profil) {
    case "TDAH":  → traitement A
    case "HPI":   → traitement B  
    case "DYS":   → traitement C
    default:      → "on ne vous reconnaît pas"
}

Problème : si tu es HPI + TDAH + femme + en réussite 
professionnelle → tu tombes dans le default. Tu n'existes pas.

Ce que le système DEVRAIT faire — des if avec conditions multiples :
if (memoire_travail < seuil 
    && vitesse_traitement > seuil
    && compensation == true
    && genre == "F") {
    → accompagnement adapté
}

### La subtilité supplémentaire :
Même dans la neurodiversité, les spécialistes font des cases.
TDA. TDAH. HPI. DYS. TSA.
Des cases plus fines. Mais toujours des cases.
C'est du switch/case avec plus de cases. Pas du if(condition).
