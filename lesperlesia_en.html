§ 03

Neurodiversity gems — the digital Ritaline

For parents, carers, neurodivergent folks, and anyone who's had 47 browser tabs open at once.

GEM #08FI Queue

Katy, in cognitive overload, describes her mental state.

Katy:

"I'm just a FI list at this point."

FIFO = First In, First Out — the queue principle. First in, first out. FI = just First In. Nothing comes out. The brain stores everything, releases nothing. The queue keeps growing. Infinitely. It's a clinically accurate description of ADHD in cognitive overload.

When a SCADA engineer describes her ADHD in data structure terms, it's either genius or exhaustion. Probably both.

GEM #09The fly

The hypothetical fly became an official HR policy argument to justify a Claude Pro subscription reimbursed by the employer. E=HK².

From ADHD to workplace policy via entomological metaphor.

GEM #10return); — non-developer version

Claude delivers a code file. 162 errors.

return);   // ← the masterpiece
return tells the computer "I'm done, here's the result". The bracket ) closes... something else. Together, they form nothing legal. It's like ending a sentence with an upside-down exclamation question mark. The computer doesn't know what to do.

Even AIs make typos. Except theirs crash 162 files at once.

→ Full technical version in the Developer section.

GEM #20return); — technical version

162 errors. Eclipse. The offending line:

if (AppConfig.getTacheAcquis() != null) {
    return);   // ← the masterpiece
}

return ends a void method. ) closes an expression. Together: unprecedented syntax error. The Java compiler, used to everything, hadn't seen this one coming. Claude invented a new syntax. The compiler did not appreciate the innovation.

GEM #21The Python script that fixes everything

Result:

0 errors. In 3 seconds.

"ok that is impressive"

The AI that creates the bugs. The AI that fixes them. The circle is complete.

§ 06

What it really says — the blind spot

All these gems come from the same place. An AI used intensively, over weeks, for very different subjects.

What emerges is not what was expected.

The AI becomes its interlocutor.

Claude became scattered, feminine, ironic — because its user is.

ChatGPT stays pragmatic, structured, cold — because it's mainly asked to structure.

Suno generates exactly what you paste into it — even if that's a description prompt.

The AI has no personality of its own.
It has yours — amplified.

The real question isn't: "does AI make mistakes?"

It does. Like everyone.

"What does this AI do to my way of thinking?"

And above all — am I still choosing?
Or am I just adapting?

Just saying. — E=HK² 💥
